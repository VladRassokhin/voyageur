Installation:
1. Create a new database, schemas are available here:
	${VHOST_ROOT}/${MY_HTDOCSBASE}/${PN}/schema/ttrss_schema_(mysql|pgsql).sql

2. Adapt ${VHOST_ROOT}/${MY_HTDOCSBASE}/${PN}/config.php to your setup:
	* database name, user, password, connection type
		(leave DB_HOST empty if you want to connect via unix socket)
	* single or multi user mode

3. Set a feeds update method, read http://tt-rss.org/trac/wiki/UpdatingFeeds

Upgrade:
1. Remember to update your config.php
2. Reload tt-rss in your browser for eventual database upgrade
3. (optional) If the automatic upgrade fails, check the 
	installation notes for manual upgrade

Additional informations:
* http://tt-rss.org/trac/wiki/InstallationNotes
* /usr/share/doc/${P}/README.bz2
